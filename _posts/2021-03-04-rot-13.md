并不是原文直接输出，而是采用了[ROT-13](https://en.wikipedia.org/wiki/ROT13)加密算法：

```python
d = {}
for c in (65, 97):
    for i in range(26):
        d[chr(i+c)] = chr((i+13) % 26 + c)
```

这是一种替换式密码，将字母旋转13位得到加密结果：

```
(i+13) % 26
```

而之所以选择13，是因为字母表只有26位，对任意字母 x 来说：

$$ROT_{13}(ROT_{13}(x)) = x$$

所以这些字母的对应关系是固定的：

![](/assets/img/ROT13-1.png)

因此这是一种对等加密的算法。`codecs` 模块已经内置了对该算法的支持，所以在 Python 里可以直接这样写：
